# Slow Rotable Vent

Parts tagged with `slow_rotable_vent` slowly rotate around their Y axis. This is a cosmetic detail — the part spins in place continuously. Only works when the "Additional Map Detail" setting is enabled.

## Setup

1. Create a `BasePart` (e.g. a vent fan blade)
2. Add the `slow_rotable_vent` tag

No attributes. The rotation speed is fixed.
